Arrange the King’s Army

Problem Statement: A king wants to arrange his army in a line following strict rules. You are given:
N → total number of soldiers in the arrangement
R → soldiers are numbered from 1 to R
end → the last soldier in the arrangement must be this number


Rules
The arrangement must start with soldier 1
The arrangement must end with soldier end
No two adjacent soldiers can have the same number
You can use any soldier number from 1 to R multiple times
 

Task: Find the total number of valid arrangements of length N.


Input Format: N, R, end


Output Format: A single integer → number of valid arrangements

Ex: 1 1 _ _ end
1 1 is adjacent so not a valid arrangement

Ex: 1 2 _ 2 end
No can be repeated so it is a valid arrangement

Note: FIRST(1) and LAST(end) pos is always fixed

Ex: 
Input
4 4 3 where N = 4(1 _ _ _), R = 3(No of soldier) and end = 3(_ _ _ 3)

Output: 7

Exp: 
So for the 2nd place, either 2,3 or 4 can come (1 2 _ 3) or (1 3 _ 3) or (1 4 _ 3), no adjacent no is there 
and for the 3rd place
1. In this case: (1 2 _ 3) either (1 2 1 3) or (1 2 4 3) are possible
2. In this case: (1 3 _ 3) either (1 3 1 3) or (1 3 2 3) or (1 3 4 3) are possible
3. In this case: (1 4 _ 3) either (1 4 1 3) or (1 4 2 3) are possible

so total = 7 combination
