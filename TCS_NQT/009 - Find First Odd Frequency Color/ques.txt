At a fun fair, a street vendor is selling different colours of balloons. He sells N number of different colours of balloons (B[]).

The task is to find the colour of the balloon which is present an odd number of times in the bunch.

Note: If there is more than one colour with odd frequency, return the first such colour in the array. Balloon colours can be uppercase or lowercase.

If all colours appear an even number of times, print: All are even


Input: 7 → Value of N
B[] = [r, g, b, b, g, y, y]
(Each element is given one per line)

Output: r
Explanation:
r → 1 time
g → 2 times
b → 2 times
y → 2 times

Only r appears an odd number of times, so output is r.
