You are given an integer N, representing the number of people initially in the Happy state.


At each iteration, people change states as follows:
1. From Happy state: 70% become Sad and 30% remain Happy
2. From Sad state: 50% remain Sad and 50% become Happy


Initial State
Happy = N
Sad = 0


Simulate the process for 4 iterations and print the final number of people in:
Happy state and Sad state


Input Format
A single integer: N (initial number of Happy people)


Output Format
Print two values: Final_Happy and Final_Sad


Constraints: 1 ≤ N ≤ 10^7


Input: 100
Output: 36.5625 63.4375


Key Formulas (each iteration)
newHappy = 0.3 * Happy + 0.5 * Sad
newSad   = 0.7 * Happy + 0.5 * Sad
