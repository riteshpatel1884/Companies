Airport security officials have confiscated several items from passengers at the security checkpoint. All the items have been dumped into a huge box (array).
Each item has a risk level: 0, 1, or 2.

You are given an array of N integers, where each integer represents the risk level of an item.

Task: Sort the array based on risk levels in ascending order.

Input"
An integer N (size of the array)
Next N lines: each line contains an integer (0, 1, or 2)

Example:
N = 7
arr = [1, 0, 2, 0, 1, 0, 2]
Output: Print the sorted array

Example Output: 0 0 0 1 1 2 2
