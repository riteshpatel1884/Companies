Write a program to manage expenses from a given total income.

Input:
Total Income: Integer value representing the total money.
For each expense, take the following inputs repeatedly until the user enters "done":
Category: String representing the type of expense (e.g., "food", "shopping").
Expense: Integer value for the amount spent.

Output:
Print the Total Income.
Print the Total Expenses (sum of all expenses).
Print the Total Savings (Total Income − Total Expenses).
Print the breakdown of expenses for each category.

Ex:
200
food
50
shopping
70
travel
30
done

Output
Total Income: 200
Total Expenses: 150
Total Savings: 50
Category Breakdown:
food: 50
shopping: 70
travel: 30
